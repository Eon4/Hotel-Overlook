## Installation

Start med at hente projektet ned
Naviger til rod-mappen
Åben en terminal og skriv følgende:
npm install
Skriv herefter:
cd app
Skriv herefter:
npm run dev

Husk at installere:
npm i sass
npm i react-hook-form
